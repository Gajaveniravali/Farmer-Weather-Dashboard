# Farmer Weather Dashboard

Simple dashboard showing weather information for farmers.

## Prerequisites
- Python 3.8+
- (Optional) Virtual environment

## Quick start

1. Create and activate a virtual environment:

```powershell
python -m venv venv
& venv\Scripts\Activate.ps1
```

2. Install dependencies (if you have a `requirements.txt`):

```powershell
pip install -r requirements.txt
```

3. Run the app:

```powershell
python app.py
```

The app typically runs on http://127.0.0.1:5000 unless configured otherwise.

## Notes
- Add a `requirements.txt` with `pip freeze > requirements.txt` to capture dependencies.
